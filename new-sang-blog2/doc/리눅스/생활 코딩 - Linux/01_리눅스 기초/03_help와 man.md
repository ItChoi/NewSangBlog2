- 명령어 메뉴얼 찾아보기!
  - 명렁어 --help
    - ls --help
  - man 명령어
    - man ls
    - 검색 키 (/)
      - /sort
      - 검색 단어 위주로 스킵 (n)
      - 빠져나갈 땐 (q)
  - help와 man 차이
    - help는 화면을 빠져나가지 않고 간단히 보여준다.
    - man은 전용 페이지로 이동 후 더 자세하게 보여준다.
    - 내용이 같을 수도 있고, 어떤 명령어에는 동작 안할 수도 있다.
- 리눅스나 유닉스에서는 앞에 .이 붙으면 숨김 파일
  - -a 속성을 쓰면 숨김 파일도 보여진다.
    - ls -a
